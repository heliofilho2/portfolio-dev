# Pasta de Arquivos de Currículo

Esta pasta contém os arquivos PDF dos currículos.

## Arquivos necessários:

1. **resume-en.pdf** - Currículo em inglês (Resume)
2. **resume-pt.pdf** - Currículo em português (CV)

## Como adicionar os arquivos:

1. Coloque os arquivos PDF nesta pasta
2. Certifique-se de que os nomes dos arquivos estão exatamente como acima
3. Os arquivos serão servidos pelos endpoints:
   - `/api/resume/en` - Download do Resume (EN)
   - `/api/resume/pt` - Download do CV (PT-BR)

## Nota:

Estes arquivos não devem ser commitados no Git (devem estar no `.gitignore`).
